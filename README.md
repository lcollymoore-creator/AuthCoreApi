# 🚀 ASP.NET Core JWT Authentication API

## 📌 Description

This project consists of building a REST API using ASP.NET Core with JSON Web Token (JWT) authentication.
The main goal is to practice implementing modern security mechanisms in backend applications, including user management, authentication, and authorization.

---

## 🎯 Objectives

* Implement JWT authentication in ASP.NET Core
* Protect endpoints using authorization
* Understand the login flow and token generation
* Apply best practices for building secure APIs

---

## 🛠️ Technologies Used

* ASP.NET Core Web API
* Entity Framework Core (optional)
* JWT (JSON Web Tokens)
* SQL Server / SQLite (optional)
* Swagger (OpenAPI)

---

## 📂 Project Structure

```
/Controllers
/Models
/DTOs
/Services
/Data
/Configurations
```

---

## 🔐 Main Features

* User registration
* Login with JWT generation
* Role-based authorization (optional)
* Endpoint protection using `[Authorize]`
* Token configuration (issuer, audience, expiration)
